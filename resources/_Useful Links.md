# Useful Links

## Choosing a language

Students can sit the AIO with either Python or C++. Python is by far the more popular option and is very simple to learn. However, there are some limitations.

If this is your first AIO, or you only want to do the competition for fun, then Python is the language for you.

On the other hand, if you are an experienced coder and you want to push towards a very very string AIO score, then learning C++ may be worth it.

If you are invited to December Camp (AMT Informatics School of Excellence), you will only be allowed to use C++ there. If you want to reach the highest level, it will pay off to learn some C++ now.
